TimeBasedWindowc back to ToC or Class ToC

IRI: http://w3id.org/rsp/vocals-sd#TimeBasedWindow

A TimeBasedWindow is a Window defined in terms of time. More formally, it is a window defined through two time instants o, c (respectively named opening and closing time instants) such that: W(S) = {d|(d, t) ∈ S ∧ t ∈ (o, c]}
has super-classes
Window c

TimeVaryingGraphc back to ToC or Class ToC

IRI: http://w3id.org/rsp/vocals-sd#TimeVaryingGraph

A TimeVaryingGraph represents and RDF graph that changes over time. Formally, A time-varying graph G is a function that relates time instants t ∈ T to RDF graphs: G : T → {g | g is an RDF graph}, where T is the time domain.
is in domain of
name
is in range of
