Introduction back to ToC

VoCaLS Core module enables stream producers to publish metadata to describe streams. It also provides a way to represent and describe finite stream partitions that facilitate historical stream processing. With metadata about the streams available on the Web, consumers can discover and select streams relevant to their tasks
